## JPG2Ascii

将图片转换成ascii图像, 推荐选择线条比较少，色彩比较单一而且颜色差距大的图片.
后端为tornado， 接受请求后使用[jp2a](https://csl.name/jp2a/)进行转换.

演示地址: [http://jpg2ascii.omem.me](http://jpg2ascii.omem.me)
